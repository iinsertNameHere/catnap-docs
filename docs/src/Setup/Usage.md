## Run catnip in you terminal:
```shell
$ catnip
```
This will use the default config files

## Change the distro icon:
```shell
$ catnip -d <distro>
```

## Check the current version:
```shell
$ catnip -v
```
This will show the commit hash your version of catnip is build on.

## Get a list of arguments:
```bash
$ catnip --help
```
